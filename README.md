# Microeconometric-analysis-corruption-logit

## Description
This project estimates a Logit model to analyze individual-level determinants of corruption exposure using survey data.

## Methodology
- Logistic Regression (Logit)
- Maximum Likelihood Estimation
- Marginal Effects
- ROC Curve
- Confusion Matrix

## Main Results
- Education and income increase probability of corruption exposure.
- Female individuals show lower probability.
- Residence in Lima increases exposure probability.
- Model shows acceptable predictive performance (ROC-AUC).

## Tools
Python, pandas, statsmodels, scikit-learn, matplotlib
